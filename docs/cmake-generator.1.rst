NAME
====

CMake-Generator - Base CMake project generator

SYNOPSIS
========

**cmake-generator** [*arg[...]*]

DESCRIPTION
===========

**CMakeGenerator** is a cli tool to generate automatically a base CMake projects for any production.

It is only usable with command line

OPTIONS
=======

**-r** **--root**
    Set project base name

**-a** **--app**
    Set application name and dependencies

**-l** **--lib**
    Set library name and dependencies
